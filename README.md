# MyCADTools

A collection of custom-built .NET applications that integrate seamlessly with the AutoCAD API to automate and streamline repetitive CAD workflows. Developed specifically for professionals in architecture, landscape architecture, and other AEC (Architecture, Engineering, and Construction) disciplines.

## 🚀 Overview

MyCADTools is designed to enhance productivity and accuracy by providing tools that simplify common CAD tasks, reduce manual input, and support consistent design documentation. Whether you're preparing drawing sets, automating annotations, or batch processing geometry, these tools aim to save time and reduce errors.

## 🔧 Features

- Seamless integration with AutoCAD via its .NET API
- Modular toolset focused on:
  - Batch operations
  - Drawing cleanup and standardization
  - Annotation and labeling automation
  - Layout and plotting enhancements
- Custom commands and workflows tailored for design professionals
- Built with extensibility and maintainability in mind

## 🏗️ Use Cases

- **Architecture:** Automate sheet setup, annotation routines, and drawing coordination.
- **Landscape Architecture:** Speed up grading plans, planting schedules, and layer management.
- **AEC Firms:** Improve consistency across projects and teams with standardized tools.

## 💻 Technologies

- **.NET Framework** (target version: 4.x)
- **C#**
- **AutoCAD .NET API**

## 📦 Installation

1. Clone or download the repository
2. Build the solution in Visual Studio
3. Load the resulting `.dll` into AutoCAD using the `NETLOAD` command
4. Run custom commands as needed
